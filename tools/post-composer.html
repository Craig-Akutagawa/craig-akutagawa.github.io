本地发帖器

这是一个只在你电脑本地运行的 Markdown 发帖器。它支持常用格式按钮、接近成品的实时预览， 以及把图片自动复制到 assets/posts/ 后再插入正文。

文章信息

常用字段放前面，摘要和 slug 收进高级选项，默认更清爽。

高级字段

Markdown 帮助

这个编辑器仍以 Markdown 为唯一真实来源，但常用格式都可以点按钮插入。

  • **粗体**、*斜体*、`行内代码`
  • # 一级标题、## 二级标题、引用、列表、任务列表
  • 插入图片时会自动复制到 assets/posts/<slug>/
  • 如果浏览器不支持本地写入，仍然可以下载 Markdown 文件

正文编辑
